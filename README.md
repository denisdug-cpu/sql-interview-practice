# sql-interview-practice
Este repositório documenta minha jornada de aprendizado em SQL, com foco em entrevistas técnicas, desenvolvimento do raciocínio lógico e resolução de problemas.

Cada exercício é resolvido seguindo um método estruturado antes da escrita da consulta SQL.

Collection of SQL interview exercises with explanations and solutions.

## Conteúdos

- SELECT
- WHERE
- IN
- DISTINCT
- JOIN
- GROUP BY
- HAVING
- ORDER BY
- COUNT
- SUM
- AVG

## Método de estudo

Para cada exercício sigo um processo de raciocínio antes de escrever o SQL:

1. Ler e interpretar a pergunta.
2. Identificar o que deve ser retornado.
3. Descobrir a tabela principal.
4. Identificar os filtros.
5. Verificar se é necessário JOIN ou IN.
6. Somente então escrever a consulta SQL.
